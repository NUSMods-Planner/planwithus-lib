Module

Module: [string, number]
  • Defined in module/types.ts:10

Type representing a module as a pair of a string and number.

The first element represents the module code, while the second element represents the number of Modular Credits (MCs).

For more information on the NUS Modular System, please refer to https://www.nus.edu.sg/registrar/academic-information-policies/graduate/modular-system.
